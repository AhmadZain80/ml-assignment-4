# Streamlit ML Dashboard

Interactive machine learning dashboard for data upload, model configuration, training, and visualization.

## Tech stack
- Python
- Streamlit
- scikit-learn
- Pandas, NumPy, Matplotlib, Plotly

## Setup
```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Run
```powershell
streamlit run app.py
```
